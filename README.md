# Getting Started

To get a local copy up and running follow these simple steps.

## Installation

1. Clone the repo

```
git clone https://github.com/PanWidmo/dishes-form.git
```

2. Install NPM packages

```
npm install
```

3. Run the dev server

```
npm start
```
